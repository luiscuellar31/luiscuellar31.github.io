# Currículum Vitae de Luis Enrique Sánchez Cuellar

Sitio estático minimalista para presentar mi CV/portafolio con enfoque en programación y proyectos destacados.

## ¿Qué incluye?
- Enlaces a mi CV, GitHub y LinkedIn.
- **Sobre mí** y secciones de proyectos, experiencia y stack tecnológico.
- Estilos modernos escritos a mano en `styles.css`.

## Uso local
```bash
# Clonar el repo
git clone https://github.com/luiscuellar31/luiscuellar31.github.io.git
cd luiscuellar31.github.io

# Abrir el sitio en tu navegador
open index.html  # macOS
# start index.html  # Windows
# xdg-open index.html  # Linux
```
